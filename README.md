# EKO Toplotne Pumpe

Dobrodošli na projekat EKO Toplotne Pumpe. Ovo je moderna prezentaciona stranica za sisteme grejanja, hlađenja i tople vode, izgrađena koristeći najnovije web tehnologije.

## O projektu

Ovaj projekat prikazuje prednosti EKO toplotnih pumpi (R32 i R290 serije), uključujući uštedu energije, ekološke aspekte i tehničke specifikacije.

## Tehnologije

Ovaj projekat je izgrađen sa:

- [Vite](https://vitejs.dev/) - Frontend tooling
- [React](https://react.dev/) - UI biblioteka
- [TypeScript](https://www.typescriptlang.org/) - Tipizirani JavaScript
- [Tailwind CSS](https://tailwindcss.com/) - CSS framework
- [shadcn/ui](https://ui.shadcn.com/) - Komponente interfejsa
- [Framer Motion](https://www.framer.com/motion/) - Animacije

## Pokretanje projekta

Da biste pokrenuli projekat lokalno:

1. Klonirajte repozitorijum.
2. Instalirajte zavisnosti:
   ```sh
   npm install
   ```
3. Pokrenite razvojni server:
   ```sh
   npm run dev
   ```

## Izgradnja za produkciju

Za kreiranje optimizovane verzije za produkciju:

```sh
npm run build
```
